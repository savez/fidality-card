PWA Open Source 100% locale

Le tue tessere fedeltà, sul tuo telefono.

Una PWA italiana che salva e condivide le tue fidelity card direttamente sul telefono. Niente account, niente server: il tuo barcode resta tuo.

Prova la demo Codice su GitHub

I tuoi dati restano sul tuo telefono.

Pensata come tool personale, non come piattaforma. Tre principi non negoziabili:

IndexedDB locale

I dati delle card vivono nel database del tuo browser. Niente cloud, niente account.

Nessun backend

L'app è 100% client-side. La condivisione viaggia nel fragment dell'URL, mai su un server.

Licenza MIT

Codice aperto. Puoi forkarlo, leggerlo, modificarlo e auto-ospitare la tua istanza.

Installala come app.

Una volta installata gira anche offline. Tre tap su qualsiasi dispositivo.

📱

Android

Chrome → menu (⋮) → "Installa app". L'icona compare in home.

🍎

iOS

Safari → condividi (□↑) → "Aggiungi a Home".

💻

Desktop

Chrome o Edge → icona di installazione nella barra degli indirizzi.
